Submission copy

Ready-to-use product description

This is the short copy to reuse in plugin listings, beta notes, and support pages.

Post Purge Pro helps WordPress site owners review stale posts before cleanup. It is designed around a conservative workflow: preview candidates, export a CSV backup, confirm the selected batch, and move content to Trash rather than permanently deleting it. The plugin is intended for maintenance providers, content-heavy sites, local business blogs, and older WordPress installations that need cleanup without risky one-click deletion.

Primary use cases

  • Reviewing old blog posts before cleanup.
  • Preparing a CSV backup for client approval or internal records.
  • Reducing stale content clutter while preserving a Trash recovery path.
  • Documenting maintenance work for client reports.

Safety boundary

Post Purge Pro should be described as a review and Trash workflow, not an irreversible deletion engine.

Privacy

The plugin workflow is site-local. CSV exports are created only when an authorized site user intentionally exports them.
