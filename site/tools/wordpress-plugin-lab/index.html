How uploading works

Drop plugin ZIPs into the artifact folder

The public page documents the workflow. Actual plugin execution happens in the private local Docker WordPress harness.

Artifact folder

local-artifacts/wordpress/

Copy each plugin ZIP here using a stable filename such as wordpress-role-auditor.zip or broken-shortcode-finder.zip.

Run the lab on Windows

npm run wordpress:lab
npm run wordpress:manifest -- wordpress/products.example.json

Install one artifact

npm run wordpress:install -- plugin local-artifacts/wordpress/example-plugin.zip --activate
npm run wordpress:install -- theme local-artifacts/wordpress/example-theme.zip
