Maxxed Remote Privacy Policy

Maxxed Remote controls compatible televisions over the user's local network. Version 1 does not create accounts, collect analytics, transmit personal data to Maxxed Technical Systems, or include an active advertising SDK.

Effective June 23, 2026

Policy scopeThe current documented Android implementation and stated release configuration.
User controlPermissions remain controlled through Android settings and in-app actions.
Contactsupport@techmaxxed.com

This page is the public privacy-policy URL for Maxxed Remote. It must be reviewed against the final signed dependency and permission report before Play Store production release.

Return to Maxxed Remote

Data processed

  • Television names or user-assigned nicknames
  • Local television IP addresses and device identifiers
  • Pairing credentials and certificate fingerprints
  • Local network state needed to discover and control a television

Permissions and purpose

InternetCommunicate with compatible televisions over the local network.
Network and Wi-Fi stateConfirm network availability and support local discovery.
Wi-Fi multicastReceive SSDP discovery responses from compatible televisions.
Nearby Wi-Fi devicesSupport local Wi-Fi discovery on Android versions that require this permission.

Storage and security

Saved televisions, pairing credentials, and trust fingerprints remain in app-private storage. Pairing secrets are protected with Android Keystore encryption.

Maxxed Technical Systems applies reasonable safeguards appropriate to the current architecture, but no device or storage system can be guaranteed completely secure.

Collection and sharing

Version 1 does not send saved television information or pairing credentials to Maxxed Technical Systems or advertising, analytics, or data-broker services.

Retention

Saved television records remain until the user removes a television, clears app data, or uninstalls the app.

Deletion

Delete individual saved televisions inside the app. Clearing Android app storage or uninstalling removes the remaining local data.

Third-party services

Television manufacturers receive commands over the user's local network according to the television's own software and privacy practices. Maxxed Remote does not proxy those commands through a developer server.

Children

Maxxed Remote is a general utility and is not designed to collect personal information from children.

Policy changes

Material changes will be published on this page with a revised effective date. A production build will not deliberately enable a new remote data service without updating its disclosures.

Contact

Questions or deletion concerns may be sent to support@techmaxxed.com.
