Rival Rush Privacy Policy

Rival Rush is a local party game. The audited internal-test source uses local player profiles, local gameplay metrics, a no-operation production analytics provider, an unavailable Google Play Games sign-in stub, and a no-operation advertising provider unless an advertising build flag and provider are deliberately enabled.

Effective June 23, 2026

Policy scopeThe current documented Android implementation and stated release configuration.
User controlPermissions remain controlled through Android settings and in-app actions.
Contactprivacy@techmaxxed.com

This page is the public privacy-policy URL for Rival Rush. It must be reviewed against the final signed dependency and permission report before Play Store production release.

Return to Rival Rush

Data processed

  • Player display names, avatar selections, settings, and local profile identifiers
  • Local wins, losses, scores, coins, experience, unlocks, challenges, and game progress
  • Local gameplay counters such as game mode opened, match started, and match completed
  • Diagnostic logs in development builds
  • Advertising ID only if a future ad-enabled production build activates an advertising SDK

Permissions and purpose

InternetReserved for Google Play delivery and any production service explicitly enabled in a future build.
Advertising IDUsed only if an ad-enabled build activates a disclosed advertising provider. The current no-operation provider does not request an ad.
NotificationsMay be used for game reminders only if the user grants permission and the feature is enabled.

Storage and security

Player identity, profile, progress, settings, rewards, and gameplay metrics are stored locally through the game's save system. The current Google Play Games provider is a non-functional stub and does not create a remote account.

Maxxed Technical Systems applies reasonable safeguards appropriate to the current architecture, but no device or storage system can be guaranteed completely secure.

Collection and sharing

The current production analytics path is no-operation and does not send gameplay events to Maxxed Technical Systems. The current advertising path is also no-operation unless an ad SDK is deliberately enabled for a later build.

Retention

Local profile and gameplay data remain until reset in the app where available, Android app storage is cleared, or the game is uninstalled.

Deletion

Clear local profile or progress controls where offered, or use Android clear-storage or uninstall to remove all app-private game data.

Third-party services

Google Play processes installation, account, purchase, and platform diagnostics under Google's terms. If Unity Ads, another ad provider, Google Play Games, Firebase, or remote analytics is enabled later, this policy and the Play Data safety declaration must be updated before that build is released.

Children

Rival Rush is intended for a general audience. The final age targeting, advertising configuration, and store declarations must be reviewed together before public release.

Policy changes

Material changes will be published on this page with a revised effective date. A production build will not deliberately enable a new remote data service without updating its disclosures.

Contact

Questions or deletion concerns may be sent to privacy@techmaxxed.com.
