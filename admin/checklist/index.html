Admin readiness checklist

Use this before switching any board from static planning rows to live operational data.

Required checks

Admin subsite prompts before page content appears.
Direct admin page URLs use the same prompt.
Admin pages are excluded from sitemap and public navigation.
Credential record is stored in 1Password.
Backend-powered boards remain disabled until the above checks pass.
