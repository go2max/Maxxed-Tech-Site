# Web Launch QA Pro

Web Launch QA Pro is a browser-local launch readiness utility for website and product-page submissions.

## Scope

- 16 weighted checks totaling 100 points
- Local project profile fields
- Readiness score and verdict bands
- Priority blockers sorted by score impact
- Markdown report copy/download
- JSON audit export/import
- Browser-local autosave through `localStorage`
- Print-friendly review output

## Validation

```bash
npm run web-launch-qa:check
npm run check
```
