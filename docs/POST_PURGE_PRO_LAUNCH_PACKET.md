# Post Purge Pro Launch Packet

## Product chosen

Post Purge Pro is the next viable launch-close option because it is already listed in the public WordPress plugin catalog and roadmap as a release candidate. It is not one of the currently active open-chat lanes for SEO Checker, Contract Extractor, Route Safe, Web Launch QA Pro, Maxxed Measure, Compass, Remote, Rival Rush, Fishing Maxxed, Gold Estimator, or the admin/site consolidation work.

## Public positioning

Post Purge Pro is a review-first WordPress cleanup plugin for stale content maintenance.

Core workflow:

1. Preview stale post candidates.
2. Export a CSV backup before making changes.
3. Confirm the intended batch.
4. Move selected content to Trash only.

The product should not be positioned as a permanent deletion engine. The launch-safe claim is that it helps WordPress operators and maintenance providers review, document, and move selected stale content to Trash with a recoverable WordPress workflow.

## Target users

- WordPress maintenance providers cleaning up old client blogs.
- Local business sites with years of thin, outdated, or duplicate posts.
- Content-heavy websites that need a reviewable stale-content workflow.
- Operators who need a client-friendly CSV record before cleanup.

## Submission description

Post Purge Pro helps WordPress site owners review stale posts before cleanup. It is designed around a conservative workflow: preview candidates, export a CSV backup, confirm the selected batch, and move content to Trash rather than permanently deleting it. The plugin is intended for maintenance providers, content-heavy sites, local business blogs, and older WordPress installations that need cleanup without risky one-click deletion.

## Short listing copy

Preview stale WordPress posts, export a CSV backup, confirm the batch, and move selected content to Trash with a review-first cleanup workflow.

## Feature bullets

- Preview stale content candidates before making changes.
- Export selected records to CSV for backup, review, or client approval.
- Require explicit confirmation before batch action.
- Move content to Trash only, preserving the normal WordPress recovery path.
- Keep the workflow inside the WordPress admin area.
- Route prerelease testing and support to support@techmaxxed.com.

## Safety and privacy notes

- Do not claim automatic legal, SEO, or compliance correctness.
- Do not claim permanent deletion.
- Do not claim cloud processing.
- CSV files are created only when an authorized site user exports them.
- Site owners remain responsible for backups, retention rules, and final review decisions.

## Remaining package stop conditions

Before plugin-directory submission, verify the actual ZIP package in a clean WordPress lab:

- Plugin activates without fatal errors.
- Admin screen is visible only to the expected capability level.
- Candidate preview loads on a demo dataset.
- CSV export includes expected identifiers and review fields.
- Confirmation is required before any batch action.
- Batch action moves selected content to Trash only.
- No irreversible delete path is exposed in the release candidate.
- Readme, screenshots, support URL, and public product page are aligned.
